## 🎲 Random Sum Generator

>A visually interactive and animated web app that generates random >mathematical problems and lets users solve them.

![HTML](https://img.shields.io/badge/HTML-5-orange)
![CSS](https://img.shields.io/badge/CSS-3-blue)
![JS](https://img.shields.io/badge/JavaScript-yellow)

---

## 🚀 Features

- 🎯 Generates random math expressions
- ➕ Supports operations: Addition, Subtraction, Multiplication, Division
- 🔢 Custom number of digits input
- 🎨 Modern UI with animations and gradients
- 👆 “Reveal Answer” button for instant solution
- 📱 Fully responsive design

---

## 🖥️ Demo

«Enter the number of digits → Click Get Random Sum → Solve → Click Reveal Answer»

---

## 🛠️ Built With

- HTML
- CSS (Advanced Animations & Gradients)
- JavaScript (Logic + Interactivity)

---

## 📂 How It Works

- User inputs number of digits
- App generates two random numbers
- Random operator is selected
- Expression is displayed
- Answer is revealed on button click
